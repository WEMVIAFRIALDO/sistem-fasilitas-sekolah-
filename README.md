# Sistem Fasilitas Sekolah

Proyek ini adalah sistem informasi manajemen fasilitas sekolah berbasis web yang dirancang untuk mengoptimalkan pendataan, pemantauan, dan pengelolaan sarana dan prasarana institusi pendidikan.

Dokumen dan aplikasi ini dikembangkan sebagai bagian dari pemenuhan tugas mata kuliah **Rekayasa Perangkat Lunak (RPL)**.

## Fitur Utama (Rencana Pengembangan)
- **Inventory Tracking:** Pendataan kondisi fisik sarana dan prasarana secara real-time.
- **Maintenance Scheduling:** Penjadwalan perawatan aset sekolah secara berkala.
- **Facility Booking:** Sistem peminjaman ruangan atau fasilitas sekolah untuk kegiatan akademik.
